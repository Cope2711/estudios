TEMA:
Abstracción

PARRAFO:
La abstracción en programación se refiere a simplificar conceptos complejos para su representación y manipulación. Esto implica centrarse en la visión externa de un objeto, separando su comportamiento específico. En la programación orientada a objetos, la abstracción permite crear modelos conceptuales de objetos o conceptos y utilizar esos modelos para crear objetos más específicos. Es el primer principio de la POO, donde la información de los elementos se simplifica para su uso.
 
TEMA:
Fetch

PARRAFO:
La función fetch() es una API de JavaScript que permite realizar peticiones HTTP para obtener recursos de forma asíncrona desde un servidor [2]. Esta API proporciona una interfaz para acceder y manipular partes del canal HTTP, como peticiones y respuestas [3]. Es comúnmente utilizada para realizar llamadas AJAX simples en JavaScript, lo que permite obtener datos del servidor sin tener que recargar la página completa [4]. El método fetch() depende del objeto window del navegador y su uso más simple implica pasar una URL para traer el contenido correspondiente [6].

TEMA:
Electron

PARRAFO:
Electron es un framework de código abierto que permite desarrollar aplicaciones de escritorio usando las tecnologías estándares de la web, por lo que se trata de una herramienta muy apropiada para los desarrolladores que tienen experiencia en la plataforma web, con Javascript y por supuesto HTML y CSS.
